⚙️ Node.js Initial Project
📌 Overview

This repository represents a foundational Node.js project designed to introduce core backend development concepts, including runtime execution, project structure, and environment configuration.

It serves as a starting point for understanding how Node.js applications are initialized, structured, and executed in a real-world development workflow.

🎯 Purpose

The objective of this project is to build a solid base in Node.js fundamentals, enabling scalable backend development through:

Understanding how Node.js runs JavaScript outside the browser
Structuring a basic project for maintainability
Executing scripts via the Node runtime
Preparing the groundwork for more advanced backend systems
🧩 Core Concepts Covered
Node.js runtime fundamentals
Entry point execution (node index.js)
Project initialization with package.json
File-based architecture
Basic script organization
🛠️ Tech Stack
Node.js — JavaScript runtime environment
JavaScript (ES6+) — Core language
NPM — Dependency and project management
📂 Project Structure
02---Node-inicial/
│
├── index.js        # Application entry point
├── package.json    # Project configuration and metadata
└── README.md       # Project documentation
⚙️ How It Works

The application is executed through Node.js, which interprets and runs the index.js file as the main entry point.

This file contains the core logic of the program and serves as the foundation for future expansion into modular or full-scale backend systems.

▶️ Getting Started
1. Clone the repository
git clone https://github.com/SebaCalvino/02---Node-inicial.git
cd 02---Node-inicial
2. Run the application
node index.js
